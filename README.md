# UO Outlands Razor for VS Code

A Visual Studio Code extension designed to make writing, reading, and maintaining Razor scripts for UO Outlands easier and more enjoyable.

This project aims to improve the scripting experience by providing editor support and tooling tailored specifically for the Razor scripting ecosystem used by the UO Outlands community.

## Features

- Syntax highlighting for Razor scripts
- Improved readability and editing experience
- Razor-specific language support
- Ongoing improvements based on community feedback

> More features are planned and contributions are welcome.

## How to use it

Create a file with `.razor` extension and open it in VSCode. Or, you can create a new file and set the language manually:

## Motivation

Razor scripting is a powerful way to automate and enhance gameplay in UO Outlands, but the tooling available for writing scripts has historically been limited.

This extension was created to provide a better developer experience inside Visual Studio Code, making it easier to write, debug, organize, and maintain Razor scripts.
